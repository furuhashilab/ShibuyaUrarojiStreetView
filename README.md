# 概要
クラウドソーシング手法を用いてオープンな360°ストリートビューデータを収集するとともに、得られた画像情報を用いた機械学習及びフォトグラメトリ技術による地物の認識と三次元点群データの生成を効率良く行い、都市のデジタルツインデータを市民と共創するコミュニティづくりを行います。
